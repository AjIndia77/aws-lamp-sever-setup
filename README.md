# AWS-LAMP-Sever-Setup
Configured an AWS EC2 Ubuntu web server using the LAMP stack (Linux, Apache, MySQL, and PHP) with Node.js LTS. Set up multiple PHP versions (7.4 & 8.1), phpMyAdmin, an FTP user for secure file management, AWS Security Groups, and SSL support for secure HTTPS communication.
What you created

Built an AWS Ubuntu Web Server with the following stack:

AWS EC2
     │
Ubuntu 24.04 LTS
     │
Apache Web Server
     │
PHP 7.4 & PHP 8.1
     │
MySQL Database
     │
phpMyAdmin
     │
Node.js LTS
     │
FTP User
     │
SSL 

This is called a LAMP Stack (Linux + Apache + MySQL + PHP) with Node.js added.

What is it used for?

This server can host PHP applications such as:

WordPress websites
Insurance portals 
Laravel applications
CodeIgniter applications
CRM systems
Admin Panels
School Management Systems
Company websites

The flow is:

Developer
      │
      │ Upload Code
      ▼
Apache
      │
PHP Executes Code
      │
MySQL Stores Data
      │
User Opens Website
